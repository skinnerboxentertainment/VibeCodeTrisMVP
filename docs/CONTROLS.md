# VibeCodeTris Controls

This document outlines the controls for playing VibeCodeTris across different input methods.

## Keyboard Controls

| Action                      | Key(s)                |
| --------------------------- | --------------------- |
| **Move Left**               | `ArrowLeft`           |
| **Move Right**              | `ArrowRight`          |
| **Soft Drop**               | `ArrowDown`           |
| **Hard Drop**               | `Space`               |
| **Rotate Clockwise**        | `ArrowUp` or `x`      |
| **Rotate Counter-Clockwise**| `z`                   |


## Gamepad Controls (Standard Mapping)

| Action                      | Button(s)             |
| --------------------------- | --------------------- |
| **Move Left**               | D-Pad Left            |
| **Move Right**              | D-Pad Right           |
| **Soft Drop**               | D-Pad Down            |
| **Hard Drop**               | `A` / `Cross`         |
| **Rotate Clockwise**        | `B` / `Circle`        |
| **Rotate Counter-Clockwise**| `X` / `Square`        |

## Touch Controls

### Virtual Buttons

The on-screen virtual buttons provide the primary touch controls:

| Action                      | Button ID             |
| --------------------------- | --------------------- |
| **Rotate Counter-Clockwise**| `btn-rot-ccw`         |
| **Rotate Clockwise**        | `btn-rot-cw`          |
| **Move Left**               | `btn-move-left`       |
| **Move Right**              | `btn-move-right`      |
| **Hard Drop**               | `btn-hard-drop`       |
| **Soft Drop**               | `btn-soft-drop`       |

### Gestures

Swipe gestures on the game container (the area with the falling blocks) provide an alternative way to move the pieces:

| Action                      | Gesture               |
| --------------------------- | --------------------- |
| **Move Left**               | Swipe Left            |
| **Move Right**              | Swipe Right           |
| **Soft Drop**               | Swipe Down            |
